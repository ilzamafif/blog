Langkah Persiapan:

  1. Siapkan VBox Data.
  2. Jika terjadi freeze, tekan ALT F2.
  3. Kapasitas 30GB tersedia.

Konfigurasi Awal:

  1. Matikan opsi ke mode biracditional.

  2. Sistem:

    • RAM: 10GB (11GB lebih disarankan).
    • Prosesor: 3 core dari total 2.
    • Memori Video: 128MB (pastikan tidak terlalu rendah).
    • Penyimpanan: Pilih SSD dan centang Solid State Drive.
    • Jaringan: Atur koneksi internet terlebih dahulu.
  3. Klik OK untuk memulai.

Instalasi Sistem:

  1. Masukkan file ISO.
  2. Unggah file.
  3. Tambahkan.

Opsi Instalasi:

  1. Pilih Instalasi Grafis.
  2. Bahasa: Pilih Bahasa Indonesia atau Inggris (disarankan).
  3. Lokasi: Sesuaikan dengan lokasi Anda.

  4. Konfigurasi:

    • Nama Komputer: Kali.
    • Kosongkan kolom Kali.
    • Password: gunakan kata sandi toor.
